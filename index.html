What is GitHub Pages?

GitHub Pages is a free and easy way to host simple, static websites directly from your GitHub repositories. It's perfect for personal portfolios, project documentation, blogs, or simple landing pages.

You can use HTML, CSS, and JavaScript, and GitHub handles the hosting and deployment automatically when you push your changes.
